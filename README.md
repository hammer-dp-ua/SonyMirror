## Обрізання. Спосіб 1.
![](/pics/2_layers.jpg)

Спочатку вирішив ничого важкого. Зараз роздрукую збільшену на 20-30% фотку дзеркальца, обріжу само дзеркальце ножицями, зовнішню частину контура роздруківки прикладу до квадратної картонки, зроблю контур олівцем на картонці та обріжу картонку по ньому. Нє-а. На картинці з 2 полупрозорими слоями 2 розмірів в Gimp гарно видно, що права частина дзеркальца не піддається такому способу. Складна геометрія не дозволяє.
