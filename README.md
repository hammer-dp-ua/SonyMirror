## Обрізання. Спосіб 1.
![](/pics/2_layers.jpg)

Спочатку вирішив ничого важкого. Зараз роздрукую збільшену на 20-30% фотку дзеркальца, виріжу само дзеркальце ножицями, наружну частину контура роздрукывки прикладу до квадратної картонки, зроблю контур олівцем на картонці та виріжу картонку по ньому. Нє-а. На картинці з 2 полупрозорими слоями 2 розмірів в Gimp гарно видно, що права частина дзеркальца не піддається такому способу. Складна геометрія не дозволяє.
